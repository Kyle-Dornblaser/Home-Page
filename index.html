Grind Bucket March 2015

About

Grind Bucket is a Ruby on Rails app for finding level caps in games. Users are automatically signed up based on some unique identifying information. Users have the ability to add games that need to be approved by an admin. A user can add a level cap to any game that has been approved. Users can vote on the level caps to assert if they are correct or not. An admin can verify a submitted level cap.

Built with: Ruby on Rails JavaScript HTML CSS PostgreSQL
